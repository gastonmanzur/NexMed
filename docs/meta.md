# Configuración pública para Meta Developers

Usar estas URLs públicas en la app de Meta:

- Meta Privacy Policy URL: `https://www.nexmedturnos.pro/privacy-policy`
- Meta Data Deletion Instructions URL: `https://www.nexmedturnos.pro/data-deletion`

La política declara que NexMed usa Meta WhatsApp Cloud API para WhatsApp Business, que pueden procesarse teléfonos, parámetros de plantillas y estados de mensajes, y que NexMed no usa esos datos para publicidad ni perfiles publicitarios.
