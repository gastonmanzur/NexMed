# WhatsApp header image

The Meta WhatsApp template header image is intentionally not committed here because the repository/PR workflow does not accept binary files.

Deploy the institutional image to this stable public path outside the source diff:

```text
/assets/whatsapp/nexmed-header.jpg
```

Production URL used by `META_WHATSAPP_HEADER_IMAGE_URL`:

```text
https://www.nexmedturnos.pro/assets/whatsapp/nexmed-header.jpg
```

The URL must remain public, stable, unauthenticated, and return `image/jpeg` or `image/png` with HTTP 200.
